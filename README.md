# 🧠 Mini Machine Learning Projects (Hands-On Learning)

Welcome!  
This repository is my personal space for practicing machine learning by **rebuilding small ML projects from YouTube tutorials**.  

Instead of just watching, I **code along, break things, fix them, and learn practically**.  
This helps me understand how real ML projects are structured and implemented.

---

## 🎯 Goals of This Repo

- Get practical ML experience beyond theory

- Learn from real examples instead of only reading
- Build a habit of hands-on learning
- Slowly transition into building my own projects
